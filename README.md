# Data Visualization and Storytelling

## Objective
Create an interactive Power BI dashboard to analyze Superstore sales data and present business insights.

## Tools Used
- Power BI Desktop
- Superstore Dataset

## Dashboard Includes
- KPI Cards
- Sales by Category
- Profit by Region
- Monthly Sales Trend
- Sales by Segment
- Top 10 Products
- Discount vs Profit
- Sales by State

## Key Insights
- Technology has the highest sales.
- West region generates maximum profit.
- Consumer segment contributes most revenue.
- High discounts reduce profitability.
- Sales increase during year-end.
